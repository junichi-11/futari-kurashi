# Release Checklist

## Editorial

- [ ] 選定理由、向いている暮らし、長所、気になる点、編集部コメントを確認した
- [ ] AI的な定型文、根拠のない断定、ランキング表現がない
- [ ] 商品画像と商品ページの対応を人間が確認した

## Data and affiliate

- [ ] Product ValidationがPASSした
- [ ] 価格、在庫、送料、レビュー、`last_verified`を再確認した
- [ ] Productionの商品数とaffiliate link属性を確認した

## Experience and delivery

- [ ] PC / Mobileで横スクロール、画像見切れ、Console Errorがない
- [ ] SEO、Schema、内部リンク、広告表記を確認した
- [ ] GitHub ChecksがPASSし、Draft PRの差分だけが対象である
