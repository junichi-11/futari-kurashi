# Brand Tokens

正式な実装値は `assets/brand/brand-tokens.css` を正本とします。

| Token | Value | Usage |
| --- | --- | --- |
| `--margin-charcoal` | `#1A1D21` | 文字、濃色背景、通常ロゴ |
| `--margin-beige` | `#C8AA8A` | ベージュ円、小さなアクセント |
| `--margin-off-white` | `#FAF9F6` | 基本背景、Reverseロゴ |
| `--margin-light-gray` | `#F2F2F2` | 補助背景 |
| `--margin-text` | `#1A1D21` | 本文色 |
| `--margin-border` | `rgba(26, 29, 33, 0.14)` | 細い罫線 |

ベージュは面で多用せず、Off White、Charcoal、薄い罫線を中心に構成します。トークン変更時はLP、記事、About、Footer、Light / Dark背景のロゴコントラストを確認します。
