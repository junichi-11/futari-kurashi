# Editorial Template

記事固有情報は `data/articles.json`、商品情報は `data/products.json` に置き、HTMLへ重複させません。

## 流用項目

- title / subtitle / description / category / tags
- heroImage / heroEditorsNote / readingTime / featuredLabel
- productIds / productGroups
- comparison.axes / comparison.values
- beforeYouChoose / closing / editorsNote
- relatedArticleIds / comingSoon / journalPath

新しい記事は記事IDをWeb Componentへ渡し、Validation、Preview、Productionの順で確認します。

参照元: `docs/Editorial-Template.md`、`data/articles.json`。最終レビュー: 2026-07-16。
